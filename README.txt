TellNo1 Static Website (multi-page)

Run locally (recommended):
- python -m http.server 8000
- open: http://localhost:8000/tellno1-site/index.html

Theme:
- Auto by OS preference (prefers-color-scheme)
- Toggle cycles Auto → Dark → Light (saved in localStorage)

Deploy:
- Upload the folder to any static host (GitHub Pages, Netlify, Vercel static, Cloudflare Pages).
